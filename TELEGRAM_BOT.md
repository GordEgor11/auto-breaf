# 🤖 Telegram Бот для Decor CRM

Этот бот помогает агентам получить их персональный Chat ID для подключения уведомлений.

## 🚀 Быстрый старт

### 1. Откройте бота

Перейдите по ссылке: **https://t.me/Build_crm_bot**

### 2. Запустите бота

Нажмите кнопку **"Запустить"** (Start) или отправьте команду `/start`

### 3. Получите Chat ID

Бот отправит вам сообщение:

```
🎉 Ваш Chat ID: 123456789

📋 Инструкция:
1. Скопируйте число выше (нажмите на него)
2. Откройте Decor CRM
3. Перейдите в Настройки → Telegram
4. Вставьте Chat ID в поле
5. Нажмите "Сохранить"

✅ После этого вы будете получать уведомления о новых заявках!
```

### 4. Сохраните в CRM

1. Откройте [Decor CRM](https://your-site.com)
2. Перейдите в **Кабинет** → **Настройки** → **Telegram**
3. Вставьте Chat ID в поле
4. Нажмите **"Сохранить"**

## ❓ Частые вопросы

### ❓ Что такое Chat ID?

**Chat ID** — это уникальный идентификатор вашего аккаунта в Telegram. По этому ID бот понимает, куда отправлять уведомления.

### ❓ Это безопасно?

Да! Бот только отправляет вам уведомления. Он не имеет доступа к вашим данным в CRM.

### ❓ Я не получил сообщение от бота

Проверьте:
1. Бот не заблокирован?
2. Вы нажали кнопку "Запустить"?
3. Попробуйте отправить `/start` ещё раз

### ❓ Как отключить уведомления?

1. Откройте CRM
2. Перейдите в **Настройки** → **Telegram**
3. Нажмите **"Отключить"**

Или просто заблокируйте бота в Telegram.

## 🔧 Для разработчиков

Бот работает на библиотеке [Telegraf](https://telegraf.js.org/).

Исходный код: `telegram-bot/index.js`

### Запуск локально:

```bash
cd telegram-bot
npm install
TELEGRAM_BOT_TOKEN=your_token npm start
```

### Запуск на Vercel:

```bash
cd telegram-bot
vercel --prod
```
